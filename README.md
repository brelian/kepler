### 线性表的实现
#### 顺序存储实现
逻辑顺序 和 物理顺一致性 ==> 使用动态数组

#### 链表存储实现
- 单链表： 只保存后续节点的指针
    头节点： 不存放数据的节点，为解决插入问题
- 双链表： 既保存前驱节点又保存后续节点
- 循环链表： 尾节点和首节点相连
