## 项目介绍

该项目致力于提高 PHP 工程师的算法基础，项目实用 PHP 定义了常用的数据结构的抽象数据类型(ADT)，训练者只需实现其中的主要逻辑即可，项目集成了 PHPunit 单元测试包，以便测试。

项目目录结构如下

```json
.
├── src
│   ├── Algorithm 			 	# 算法
│   │   ├── Search 				# 搜索算法
│   │   └── Sort				  # 排序算法
│   └── Structure
│       ├── ADT					  # 抽象数据类型
│       ├── Heap				  # 堆
│       ├── LinkedList	  # 链表
│       ├── PriorityQueue # 优先队列
│       ├── Queue				  # 队列
│       ├── Stack			  	# 栈
│       └── Tree				  # 树
└── tests						      # 测试用例

```
