逻辑顺序 和 物理顺一致性 ==> 使用动态数组

#### 链表存储实现
- 单链表： 只保存后续节点的指针
    头节点： 不存放数据的节点，为解决插入问题
- 双链表： 既保存前驱节点又保存后续节点
- 循环链表： 尾节点和首节点相连

### 平衡二叉树
### 概念
- 每个节点的平衡因子都属于 {-1,0,1} => 左右子树高度最多相差 1
- 平衡因子： 节点的左子树高度 - 右子树高度
- 平均操作： O(logN)

#### 红黑树
### 定义
1. 每个节点非红即黑
2. 根是黑色
3. 一个节点是红色那么它的孩子节点都是黑色
4. 如果把叶子节点定义为空域节点的话，任何节点到可达的叶子节点之间的黑色节点数量相同。
### 特性
- 最坏操作： O(logN)
